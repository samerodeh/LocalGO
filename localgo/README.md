# LocalGo — Landing Page

Fast & Reliable Delivery · Montreal, QC

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
npm run preview
```

## Stack

- React 18 + TypeScript
- Vite 5
- No external UI frameworks
- Google Fonts: Barlow + Barlow Condensed
